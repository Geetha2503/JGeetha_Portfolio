UG-Projects



LED Propeller Clock

Built a physical prototype of an LED-mounted propeller that displays the time in digital format, i.e. XX:XX:XX - hour:minutes:seconds. To generate a circular screen for display, a linear array of LEDs is mounted on a DC motor rotating at 1000 rpm. Using Arduino programming, I optimized the angular velocity of the propeller to achieve a clear display of characters.

P10 LED Scrolling

The digital scrolling message display that I created can display digital texts on a board made of an LED array. Through pre-programming the data that it displays on an Arduino UNO microcontroller. Even though this control card has a wide range of shapes and features, the Arduino DMD library enables us to programme the Arduino board to communicate with the P10 dot matrix display.

Smart Serving Robot

Developed a compact portable WiFi module based wireless serving robot withNodeMCU as central driving unit. A camera is fixed in the head of the robot to make wireless 2 way communication with serving environment. Created Mobile application to serve the purpose of controller to the robot using MIT App Inventor. It is recognised as the best project among all (~30) the final year projects of ECE department.

Sensor Tap

This is a physical prototype that is completely functional, and hotels and restaurants are urged to keep it that way. A significant portion of circuit design was carried out using external PCB design techniques and deep learning with regard to the sensors. We used the Arduino simulator to simulate the entire project before using various PCB designing techniques to turn it into hardware. Additionally, an ultrasonic sensor was installed with a transmitter and receiver housed in two separate chambers. The transmitter sends out continuous sound waves, and the receiver receives signals from the receiver if anything obstructions the range of the sensor, such as a hand, are detected. which requires coding in order to dump the code to an Arduino UNO micro controller.
